# Image

The <xref:System.Windows.Controls.Image> element displays bitmap images in Windows Presentation Foundation (WPF) applications. Use this control to show various image formats and manipulate their appearance through transforms like cropping, rotating, and converting to greyscale.

| [Convert an Image to Greyscale](how-to-convert-an-image-to-greyscale.md) | Learn how to convert a color image to a greyscale image. |
| [Crop an Image](how-to-crop-an-image.md) | Learn how to crop an image. |
| [Rotate an Image](how-to-rotate-an-image.md) | Learn how to rotate an image by 90 degrees. |

## Styles and templates

The <xref:System.Windows.Controls.Image> control doesn't define template parts or visual states. It's a simple display element that renders bitmap content.

### Content property

This control doesn't define a content property.
